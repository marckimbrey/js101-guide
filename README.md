#JS101 Road Map
A terminal guide for learning javascript with the Bristol JS101 meet up group.
   
To be release as a npm module when complete.t

## Usage

```
git clone git@github.com:rinse0ut/js101-roadmap.git
cd js101-roadmap
npm install
npm start
```
